# Prototype_2023BTS

2023 실전문제연구 프로젝트 - 용접 강관 이상탐지 플랫폼의 실증을 위한 Streamlit 프로토타입 입니다.

## 페이지 구성

- Upload Image.py
- Data Preview.py
- Data Augmentation.py
- Train Model.py
- Evaluation.py
- Inference Segmentation Model.py

## 사용 방법

```
docker build {dockerfile path} -t bts2023
docker run {options} bts2023
```
